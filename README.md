# Teradici

Install NPM Dependency

### `npm install`

Run Project

### `npm install && npm run start`

Run Test

### `npm run test`

Test Coverage Report

### `npm run coverage`

# Docker

Build Docker Container

### `docker build . -t local/teradici`

Run Docker Container

### `docker run -p 3000:3000 -d local/teradici`
